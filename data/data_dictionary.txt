DATA DICTIONARY - DATASET SINTETIS UMKM KABUPATEN SLEMAN (v6)

File: umkm_synthetic_sleman_v6.csv
Deskripsi: Dataset ini merupakan data sintetis yang meniru profil demografi dan ekonomi 
UMKM di Kabupaten Sleman untuk keperluan pemodelan Machine Learning.

DAFTAR KOLOM:

1.  kapanewon              : Nama kecamatan (kapanewon) di Kabupaten Sleman tempat UMKM berada.
2.  sektor_usaha            : Kategori industri UMKM (Agribisnis, Jasa, atau Perdagangan).
3.  skala_usaha             : Tingkatan skala usaha berdasarkan kriteria aset/omset (Mikro Lv1 s/d Menengah).
4.  omset_per_bulan         : Nilai pendapatan kotor rata-rata UMKM dalam satu bulan (Rupiah).
5.  omset_pct_dalam_skala   : Posisi relatif omset UMKM di dalam rentang skalanya (Nilai 0.0 s/d 1.0).
6.  tenaga_kerja_total      : Jumlah total personil/karyawan yang bekerja di UMKM tersebut.
7.  tenaga_kerja_laki       : Jumlah karyawan berjenis kelamin laki-laki.
8.  tenaga_kerja_perempuan  : Jumlah karyawan berjenis kelamin perempuan.
9.  punya_whatsapp          : Status kepemilikan WhatsApp Business (1 = Ya, 0 = Tidak).
10. punya_instagram         : Status kepemilikan akun Instagram untuk bisnis (1 = Ya, 0 = Tidak).
11. punya_website           : Status kepemilikan website resmi atau landing page (1 = Ya, 0 = Tidak).
12. digital_score           : Akumulasi skor adopsi platform digital (0-3).
13. is_go_online            : Target Variable. Status transformasi digital UMKM (1 = Go Online, 0 = Belum).
14. is_urban                : Klasifikasi wilayah Kapanewon (1 = Kawasan Perkotaan, 0 = Kawasan Pedesaan).
15. umur_usaha              : Durasi operasional UMKM sejak berdiri hingga saat ini (Tahun).

Catatan Teknis:
Dataset ini dibuat menggunakan teknik Weighted Random Sampling berdasarkan data rekapitulasi resmi Dinas Koperasi dan UKM Kabupaten Sleman tahun 2026.
